# 客户端

Aleo 网络中的客户端是一个节点，它提供区块链数据并与网络交互，但不参与共识或区块生产。客户端在生态系统中扮演着至关重要的角色，通过提供对区块链数据的访问和促进用户与网络的交互。

## Aleo 客户端的关键功能

1. **数据检索**：客户端可以获取和提供区块链数据，包括区块、交易和状态信息，供用户和应用程序使用。

2. **交易提交**：用户可以通过客户端节点向网络提交新的交易。

3. **网络交互**：客户端与其他节点保持连接，以了解最新的区块链状态。

4. **轻量级操作**：与验证者和证明者相比，客户端的硬件要求较低，使其更容易运行。

5. **API 访问**：许多客户端暴露 REST API，允许开发人员构建与 Aleo 网络交互的应用程序。

## 运行 Aleo 客户端节点

### 系统要求
- **操作系统**：Ubuntu 22.04 (LTS)、macOS Sonoma 或更高版本、Windows 11 或更高版本
- **CPU**：32 核
- **内存**：32GB RAM
- **存储**：300GB 磁盘空间（PCIe Gen 3 x4 NVME SSD 或更好）
- **网络**：100Mbps 上传和下载带宽

### 启动客户端节点

通过运行以下命令启动客户端节点：
```bash
./run-client.sh
```

或者使用以下命令进行更多控制：
```bash
snarkos start --client
```

通过运行客户端节点，您为 Aleo 网络的去中心化和健壮性做出贡献，同时直接访问其数据和功能。

要更快地引导您的节点，请参阅账本快照指南。
